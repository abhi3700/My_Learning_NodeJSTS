# Todo App

## Description

- Built using Vite + React using TS.

> NOTE: You can also choose NextJS instead of Vite.

## Getting Started

1. Create a vite project.

```sh
bun create vite@latest
```

Choose:

- framework: **React**
- variant: **TS + SWC**

2. Install dependencies.

```sh
bun install
```

3. Copy `styles.css` file from the YTuber's repo.
4. Add prettier.
